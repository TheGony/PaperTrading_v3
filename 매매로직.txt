1. 전략 개요
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

두 가지 전략이 독립적으로 동시 운영됩니다.

  ORB (Opening Range Breakout)
    - 갭상승 종목 대상, 09:05~09:30 진입 윈도우
    - 09:00~09:04 봉으로 Range 확립 후 상단 돌파 시 매수

  MOMENTUM
    - 거래량·등락률·RSI 기반 장중 모멘텀 종목
    - 09:00 이후 전 시간대 진입 허용 (단, 15:20 이후 신규 진입/선정 금지)

[후보 선정 — orb_selector.py]

  사용 API (병렬 조회):
    ka10032  거래대금 상위 30종목
    ka10030  당일 거래량 상위 30종목
    ka10027  전일대비 등락률 상위 30종목  → 보조 데이터
    ka10029  예상체결 등락률 상위 30종목  → 보조 데이터

  풀 구성:
    ka10032 + ka10030 병합 (중복 시 ka10030 우선), ETF/ETN 제외

  캔들 필터 (09:00~09:10 봉 기준):
    - 갭 상승 (당일 시가 > 전일 종가) 종목만
    - 갭 하락(gap <= 0) 제외
    - 시가 대비 현재가 -2% 이상 눌림 제외

  소프트 패널티 (스코어 감산):
    - 갭 범위 1.5~8% 벗어남: -0.2
    - 상단 꼬리 비율 > 5%:    -0.2
    - 음봉 2개 이상:           -0.2

  스코어 공식:
    score = trde_amt_norm   × 0.25   (거래대금, log 정규화)
          + volume_norm     × 0.30   (당일 거래량, log 정규화)
          + exp_flu_rt_norm × 0.25   (예상체결 등락률)
          + flu_rt_norm     × 0.10   (등락률)
          - penalty

  최대 후보: 15종목 (ORB_CANDIDATES_MAX)
  최소 보장: 5종목 (캔들 필터 미통과 시 거래대금 순 보충)


[진입 조건 — entry.py _try_orb_entry]

  ORB Range 확립:
    - 09:00~09:04 1분봉 3개 이상 필요
    - orb_high = 해당 구간 최고가
    - orb_low  = 해당 구간 최저가

  진입 필터:
    1. 갭상승 (gap_up = True) 확인
    2. 현재가 > orb_high (돌파 발생)
    3. 현재가 ≤ orb_high × 1.01 (추격매수 방지)
    4. 현재 거래량 ≥ 직전봉 거래량 × 1.5
    5. RSI 50 이상 75 이하
    6. ORB 매수 횟수 < orb_max_count (기본값 5)
    7. 쿨다운: 동일 종목 매도 후 20분 경과

  돌파 유지 확인:
    - 3초 동안 0.5초 간격으로 현재가 > orb_high 확인

  손절 기준:
    - orb_stop_pct = max(orb_low / 진입가 - 1, -2.0)
    - ORB 저점 이탈 또는 -2% 중 타이트한 쪽 (더 높은 가격)

  진입 시간: 09:05 ~ 09:30

[종목 선정 — selector.py _fetch_momentum_stocks]

  사용 API (병렬 조회):
    ka10030  당일 거래량 상위 30종목  (주)
    ka10023  거래량 급증 종목 30종목  (보조)
    ka90009  외인·기관 매수 종목

  1차 필터:
    - ETF/ETN/레버리지/인버스 등 제외
    - 등락률 > 23% 과열 제외
    - 등락률 ≥ +0.3% AND 거래대금 ≥ 10억원 (trde_amt ≥ 1000, 백만원 단위)

  차트 필터:
    - RSI 14봉 계산 (데이터 부족 시 제외)

  스코어 공식:
    score = volume_norm   × 0.35   (거래대금, log 정규화)
          + flu_norm      × 0.30   (등락률 min-max 정규화)
          + rsi_norm      × 0.20   (RSI / 100)
          + foreign_score × 0.15   (외인·기관 매수 여부, 1.0 or 0.0)

  선정 종목 수: stock_count (설정값, 기본 10)

  갱신 주기:
    - 초기 선정: 09:01 (장 시작 직후)
    - 주기 갱신: 매 5분 (STOCK_REFRESH_INTERVAL = 300초)
    - 즉시 보충: 동일 종목 당일 손실 2회 발생 → 즉시 제거 후 다음 루프에서 보충
    - 갱신 방식: 점수 기반 선택적 교체 (기존 종목 유지 점수와 신규 후보 점수 비교)


[진입 조건 — entry.py _check_charts_and_trade]

  기준봉: 직전 5봉(1분봉) 고점 돌파 (breakout_bars = 5)

  진입 필터:
    1. 현재가 > breakout_high (고점 돌파)
    2. 현재가 ≤ breakout_high × 1.015 (추격매수 방지)
    3. RSI 45 이상 70 이하
    4. RSI ≥ 직전 RSI × 0.98 (RSI 상승 중)
    5. 당일 동일 종목 손실 2회 미만
    6. 등락률 ≤ 23% (과열 제외)
    7. 고점 근접 필터: 현재가 ≥ 당일 최고가 × 0.98 AND 거래량 ≤ 직전봉 × 1.7 시 스킵
    8. 쿨다운: 동일 종목 매도 후 20분 경과
    9. is_entry_allowed(): 15:20 이후 진입 금지

  돌파 유지 확인:
    - 3초 동안 0.5초 간격으로 현재가 > breakout_high 확인

[데드크로스 청산 — entry.py (매 1분 체크)]
  - MA5 < MA20 데드크로스 AND RSI < 45 → 매도

[수익률 기반 청산 — exit.py (매 1초 체크)]

  조기 손절:
    - 진입 후 2분 이내 수익률 < -1.2% → 즉시 매도

  ORB 전용 (strategy == 'ORB'):
    - ORB 손절: pl_rt ≤ orb_stop_pct (= max(저점기준, -2%))
    - 수익 반납 방지: peak ≥ 1.5% 도달 후 pl_rt < 0% → 즉시 매도
    - 동적 트레일링 (4단계):
        peak <  2%: trail = 1.2%
        peak <  4%: trail = 1.5%
        peak <  7%: trail = 2.0%
        peak >= 7%: trail = 2.5%

  MOMENTUM 전용 (strategy == 'MOMENTUM'):
    - 고정 손절: pl_rt ≤ effective_stop
        장초반(early):  -2.0%
        장후반(late):   stop_loss_rate 설정값
        그 외:          stop_loss_rate 설정값
    - 동적 트레일링 (4단계):
        peak >= 7%: trail = 1.5%
        peak >= 4%: trail = 2.0%
        peak >= 2%: trail = 2.5%
        peak <  2%: trail = effective_trail (장초반 2%, 후반 2.5%, 중반 trailing_stop_gap 설정값)

  트레일링 발동 조건: peak > 0% AND pl_rt ≤ peak - dynamic_trail

6. 매수 조건 (공통 — entry.py _buy_stock)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  매수 금액 = 총자산 × (buy_ratio / 100)
  매수 수량 = int(매수금액 / 현재가)
  총자산   = 예탁자산평가액 + 주식평가금액
